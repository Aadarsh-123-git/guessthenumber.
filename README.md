# 🎮 Guess The Number

A modern and interactive **Guess The Number** game built using **Python (Flask)** for the backend and **HTML, CSS, and JavaScript** for the frontend.

This project demonstrates full frontend–backend integration, dynamic API communication using `fetch()`, and real-time DOM updates without reloading the page.

---

## 🚀 Features

- 🎯 Interactive Web-Based Gameplay  
- 🌐 Flask Backend with REST API  
- ⚡ Dynamic JavaScript (No Page Reload)  
- 🎨 Responsive and Clean UI Design  
- 💻 CLI Version Included  
- 🔐 Bonus Password Generator  
- 📁 Organized Project Structure  

---

## 🛠 Technologies Used

- Python 3  
- Flask  
- HTML5  
- CSS3  
- JavaScript

## 📦 Installation & Setup

1. Clone the repository:
   git clone https://github.com/Aadarsh-123-git/guessthenumber.git  
   cd guessthenumber  

2. Install dependencies:
   pip install -r requirements.txt  

3. Run the application:
   python app.py  

4. Open your browser and visit:
   http://localhost:5000  

---

## 🎮 How It Works

1. Download the index.html file
2. open the file
3. Click **Start Game**  
4. The server generates a random number  
5. Enter your guess  
6. The system responds with:
   - Too High 🔺  
   - Too Low 🔻  
   - Correct 🎉  
7. The game updates instantly without refreshing
